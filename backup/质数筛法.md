要知道质数筛法，我们先清楚的认识到什么是质数。

质数：又叫素数，是指在大于1的自然数中，除了1和它本身之外不再有其他因数的自然数。

明白这个之后判断质数的代码显然比较清楚了：
```cpp
bool check(int x)
{
    if(x<2) return false;
    for(int i=2;i*i<=x;i++)
    {
        if(x%i==0) return false;
    }
    return true;
}
```
注意i*i部分容易爆int。

那么接下来解释质数的筛法：

埃氏筛
思路：从2开始遍历，如果当前数没有被标记为合数，它就是质数，然后将它的倍数都标记为合数。
时间复杂度：O（NloglogN）
代码：
```cpp
ll p[N];
ll cnt = 0;
bool st[N];
void init()
{
    for (int i = 2; i < N; i++)
    {
        if (!st[i])
        {
            p[++cnt] = i;
            for (int j = i * i; j < N; j += i)
            {
                st[j] = true;
            }
        }
    }
}
```
注意写博客才发现之前写埃式筛的时候喜欢j从i+i或者直接从i开始，但是这样会重复一些，我们可以从i*i开始筛。核心论点是：对于任意一个合数 x，若其最小质因子为 mp，则 x 一定会在遍历到 mp 时被标记。虽然这样也不能保证完全不重复，也算一小段优化思想，值得去理解。

线性筛
为了解决埃氏筛中同一个合数会被多个质因数重复标记的问题，线性筛做到了每个合数只被其最小的质因子筛掉一次。
时间复杂度：严格的O（N）
思路：当遍历倒i%p[j]==0时break。因为此时p[j]是i的最小质因子，如果继续往后用p[j+1]*i,得到合数i*p[j+1]的最小质因子其实是p[j],它应该在后续遍历倒某个数时被p[j]筛掉，从类保证不重不漏。

代码：
```cpp
ll p[N];
ll mn[N];
bool st[N];
ll cnt = 0;
void init()
{
    for (int i = 2; i < N; i++)
    {
        if (!st[i])
        {
            p[++cnt] = i;
            mn[i] = i;
        }
        for (int j = 1; j <= cnt && p[j] * i < N; j++)
        {
            st[i * p[j]] = true;
            mn[i * p[j]] = p[j];
            if (i % p[j] == 0)
                break;
        }
    }
}
```
区间筛法：
当你面对的数据很大的时候，但区间长度[L,R]较短时，普通的筛法会因为数组越界而失效，区间筛法利用了“合数必有不大于根号R的质因子”这一性质，实现了空间复杂度的极大优化。

思路：
1.预筛：先筛出[2,根号R]范围内的所有质数，存入p数组。
2.映射：将区间[L,R]映射到st数组的[0,R-L]上，st[j-L]代表j是否为合数。
3打击：遍历预筛除的所有质数p[i]，计算出他们在[L,R]区间内的第一个倍数start，然后以p[i]为步长，将区间内的合数全部标记为true。

代码：
```cpp
ll p[N];
ll cnt = 0;
bool st[N];
bool st_range[N];
void init_prime(ll n)
{
    mem(st, 0);
    cnt = 0;
    for (ll i = 2; i <= n; i++)
    {
        if (!st[i])
        {
            p[++cnt] = i;
            for (ll j = (ll)i * i; j <= n; j += i)
            {
                st[j] = true;
            }
        }
    }
}

void init(ll L, ll R)
{
    init_prime(sqrt(R) + 1);
    memset(st_range, 0, sizeof(bool) * (R - L + 1));
    if (L == 1)
        st_range[0] = true;
    for (int i = 1; i <= cnt; i++)
    {
        ll prime = p[i];
        ll start = max(prime * prime, (L + prime - 1) / prime * prime);
        for (ll j = start; j <= R; j += prime)
        {
            st_range[j - L] = true;
        }
    }
}
```
